win32gui.AngleArc

AngleArc(hdc, Y, Y, Radius, StartAngle, SweepAngle)

Draws a line from current pos and a section of a circle's arc

Parameters

hdc : PyHANDLE

Handle to a device context

Y : int

x pos of circle

Y : int

y pos of circle

Radius : int

Radius of circle

StartAngle : float

Angle where arc starts, in degrees

SweepAngle : float

Angle that arc covers, in degrees
