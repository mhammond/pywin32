win32ui.CreateFont

PyCFont = CreateFont(properties)

Creates a PyCFont object.

Parameters

properties : dict

A dictionary containing the font properties. Valid dictionary keys are:
height
width
escapement
orientation
weight
italic
underline
strike out
charset
out precision
clip precision
quality
pitch and family
name

Comments
