win32gui.TransparentBlt

TransparentBlt(Dest, XOriginDest, YOriginDest, WidthDest, HeightDest, Src, XOriginSrc, YOriginSrc, WidthSrc, HeightSrc, Transparent)

Transfers color from one DC to another, with one color treated as transparent

Parameters

Dest : PyHANDLE

Destination device context handle

XOriginDest : int

X pos of dest rect

YOriginDest : int

Y pos of dest rect

WidthDest : int

Width of dest rect

HeightDest : int

Height of dest rect

Src : PyHANDLE

Source DC handle

XOriginSrc : int

X pos of src rect

YOriginSrc : int

Y pos of src rect

WidthSrc : int

Width of src rect

HeightSrc : int

Height of src rect

Transparent : int

RGB color value that will be transparent
