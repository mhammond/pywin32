PyCToolBarCtrl.SetBitmapSize

SetBitmapSize(width, height)

Set the size of the actual bitmapped images to be added to a toolbar control.

Parameters

width=16 : int

Width of bitmap images.

height=15 : int

Height of bitmap images.

Alternative Parameters

width

Width of bitmap images.

height

Height of bitmap images.
