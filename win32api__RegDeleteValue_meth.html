win32api.RegDeleteValue

RegDeleteValue(key, value)

Removes a named value from the specified registry key.

Parameters

key : PyHKEY/int

An already open key, or any one of the following win32con constants:
HKEY_CLASSES_ROOT
HKEY_CURRENT_USER
HKEY_LOCAL_MACHINE
HKEY_USERS

value : string

The name of the value to remove.
