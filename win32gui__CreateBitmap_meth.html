win32gui.CreateBitmap

PyGdiHANDLE = CreateBitmap(width, height , cPlanes , cBitsPerPixel , bitmap bits )

Creates a bitmap

Parameters

width : int

bitmap width, in pixels

height : int

bitmap height, in pixels

cPlanes : int

number of color planes

cBitsPerPixel : int

number of bits to identify color

bitmap bits : None

Must be None
